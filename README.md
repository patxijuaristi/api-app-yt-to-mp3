# YouTube to MP3

API and Flutter app to download MP3 files from YouTube videos.

## Structure

- `api/` - FastAPI backend
- `app/` - Flutter app

## Requirements

- Python 3.10+
- Flutter 3.x
- FFmpeg

## Installation

### API
```bash
cd api
pip install -r requirements.txt
python main.py
```

### App
```bash
cd app
flutter pub get
flutter run
```
